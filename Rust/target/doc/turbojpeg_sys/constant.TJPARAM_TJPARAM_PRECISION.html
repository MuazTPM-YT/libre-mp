Data precision (bits per sample)

The JPEG image uses (decompression) or will use (lossless compression) the specified number of bits per sample. This parameter also specifies the target data precision when loading a PBMPLUS file with #tj3LoadImage8(), #tj3LoadImage12(), or #tj3LoadImage16() and the source data precision when saving a PBMPLUS file with #tj3SaveImage8(), #tj3SaveImage12(), or #tj3SaveImage16().

The data precision is the number of bits in the maximum sample value, which may not be the same as the width of the data type used to store the sample.

Value

  • 8 or 12 for lossy JPEG images; 2 to 16 for lossless JPEG and PBMPLUS images

12-bit JPEG data precision implies #TJPARAM_OPTIMIZE unless #TJPARAM_ARITHMETIC is set.
